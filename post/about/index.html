🏠 关于本站

比起微信公众号（恐惧与爱的行为），我希望有更稳定、安全、公开、全面的个人平台，因此利用 GitHub Pages 和 Gridea 配置了这个静态网站。

站名来自我被封的微博账号名：0x5d5d5d5d，一个我至今没有找到的野指针。

本站会不定期地以文章的形式发布我的：

  • 汉语新诗作品
  • 摄影作品
  • 代码项目文档
  • 音乐评论、推荐
  • 任何想说的

今天起，本站会作为我的个人存档，所有的想法和作品会第一时间记录在这里。微信公众号则倾向于呈现一些精选的内容。

目前正在进行一些文档转移工作，稍后可以看到比较详细的作品归档。感谢大家！
